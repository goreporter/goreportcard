Features

Summary

Including project evaluation scoring, project name, unit test coverage, and package coverage with unit tests, code simplifies hints, code redundancy, and high-circle complexity hints. Different scoring will have different expressions.

Unit Test

Unit test is the most important step for a successful project.So the more high code cover,the more safe.Goreporter calculated and counted the code coverage of the packages and the time-consuming.

Simple Code

Some code is optimized, where the package will be given as a unit.Not only will give the location of the code can be optimized, but also given the optimization program.

Function Cyclo

The function circle complexity is a criterion for evaluating the quality of a function, and the function of the circle complexity value greater than 15 is something that should be of concern and should be optimized.

Copy Code

In large projects, the repetition of code is often seen, we should pay attention to this situation, and as much as possible to do code reuse.
